# PropertyPulse 

AI-powered property price prediction and investment analysis system.

## About Project

PropertyPulse is a Machine Learning based real estate valuation web application.
It predicts property prices based on location, area, bedrooms, bathrooms and property age.

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Features

- Property price prediction
- Location-based analysis
- Investment score calculation
- Web-based user interface
